myFlix API

The backend of an app that stores information about movies and user profiles so users can utilize it to look up details about different movies.

Movie and user data are stored in a MongoDB database, and this REST API interacts with the database to allow users to retreive movie details, authenticate themselves against the user database, and update their user profiles including saving and removing movies from their list of favorites.

Technologies: JavaScript, Express, Node.js, MongoDB, Mongoose, body-parser, express-validator, passport, Postman, Heroku
