Recipe App (Web App Version)

A web application version of a recipe app, which allows registered users to create, view, modify, and delete recipes. Users can also search for recipes by recipe name and/or ingredient, as well as see graphic representation of the filtered recipes - displaying the cooking time by recipe and the allocation of the recipes' difficulty levels. This application has multiple users and allow users to view the profile of the user who created a certain recipe and view all recipes by that user.

This version of the recipe app is a fully fledged web application with multiple users and an admin panel, built using the Django web framework. It utilizes a Postgres database hosted through Koyeb and a third party app to serve media files.

Technologies: Python, Django, virtualenvwrapper, pip, SQLite3, PostgreSQL, pandas, matplotlib, Cloudinary, Koyeb, WhiteNoise

myFlix API

The backend of an app that stores information about movies and user profiles so users can utilize it to look up details about different movies.

Movie and user data are stored in a MongoDB database, and this REST API interacts with the database to allow users to retreive movie details, authenticate themselves against the user database, and update their user profiles including saving and removing movies from their list of favorites.

Technologies: JavaScript, Express, Node.js, MongoDB, Mongoose, body-parser, express-validator, passport, Postman, Heroku
